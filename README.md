# AI Code Review Assistant ⚡

A free AI-powered tool that helps students debug code instantly.

## Live Demo
[Try it here](https://code-review-app-self.vercel.app/)

## Features
- Syntax highlighted code editor (CodeMirror)
- Supports Python, JavaScript, Java, C++, HTML, CSS
- Beginner-friendly bug explanations
- Fixed code suggestions
- Review history
- Beginner / Intermediate / Advanced modes

## Tech Stack
- Frontend: HTML, CSS, JavaScript, CodeMirror
- Backend: Node.js, Express
- AI: Groq API (Llama 3.3 70B)
- Deployment: Vercel (frontend) + Render (backend)

## Run locally
```bash
git clone https://github.com/raz-88/code-review-app
cd code-review-app
npm install
# Add your GROQ_API_KEY to .env
npm start
```
